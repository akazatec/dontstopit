📌 About the Tool
Dontstopit V4.0 is an educational automation tool that demonstrates how repeated messaging requests are handled by APIs.

This project is focused on:

Learning automation flow
Cybersecurity awareness
Ethical hacking practices
✨ Features
Terminal-based interface
Clean & beginner-friendly code
Cross-platform support
API-based demonstration
No telecom balance usage
